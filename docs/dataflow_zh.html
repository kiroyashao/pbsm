PBSM 数据流

预测 → 验证 → 纠正 — 认知循环

输入管线

  • • 原始输出→ToolAdapter→BeliefGraph
  • • 接收 JSON、CSV、ERROR、TEXT 格式
  • • 解析为结构化断言
  • • 更新信念节点和边
  • • 从工具输出创建新信念

核心循环

  • • 预测→验证→纠正循环
  • • 从信念图生成预测
  • • 对观察计算残差
  • • 更新置信度（验证 +0.05）
  • • 降低置信度（证伪 -0.15）

反馈

  • • 残差驱动信念更新
  • • 异常触发元认知调整
  • • 漂移检测与检查点恢复
  • • 高残差时注意力重分配
  • • 存储经验以供后续检索
